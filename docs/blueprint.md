# **App Name**: Maromba Moonshine

## Core Features:

- Homepage Banner: Display a rotating banner with three images of drink kits, with subtle glitch and smoke effects.
- Kit Display: Showcase available kits with names, dark/alien images, descriptions, and prices.
- One-Click Purchase: Implement a direct checkout process with no cart; clicking 'Buy Now' immediately redirects to the checkout page.
- Alien-Themed Checkout: Design a checkout page with a secret alien base theme, including a form for customer details.
- Payment Integration: Set up payment processing for Pix and credit cards with thematic but clear language.
- Data Storage: Store kit, order, and customer data in Firebase.
- Order Management: Track orders including kit details and customer data for shipping using Firestore.

## Style Guidelines:

- Primary color: Dark purple (#301934) to convey mystery and a premium feel, aligning with the alien darkness theme. This choice complements both light and dark elements, allowing for flexible design implementations without feeling cliched or generic.
- Background color: Deep black (#000000) to enhance the night-time atmosphere. This ensures all lighter design elements such as the alien green and metallic gray, as well as text, are clearly visible and stand out.
- Accent color: Neon alien green (#39FF14) for glowing highlights and interactive elements.
- Headline font: 'Space Grotesk', a sans-serif font with a techy, scientific feel. Body font: 'Inter', for clear readability.
- Use dark icons with neon green glows to maintain the alien aesthetic.
- Create a dark, immersive layout with strong shadows and neon glow effects.
- Implement subtle glitch effects and smooth transitions to enhance the user experience.
